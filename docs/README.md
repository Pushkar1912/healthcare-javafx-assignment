# Healthcare Project Documentation

Documentation for the Healthcare Management System (RMIT COSC1295 Advanced Programming, Assignment 2). This folder contains everything needed to understand and work with the project.

## What's in Here

### [database-overview.md](./database-overview.md)
Explains what the database does and how all the tables work together. Good for understanding the data structure without getting lost in technical details.

### [setup-guide.md](./setup-guide.md)
Step-by-step instructions for getting the project running, for both Mac and Windows.

### [testing-guide.md](./testing-guide.md)
Everything about the testing setup - what tests exist, what they check, and how to run them.

### [report.md](./report.md)
A reflective report covering:
- Technology challenges faced
- Better alternatives discovered
- Design decisions and lessons learned
- What I'd do differently next time

## Quick Start

1. **New to the project?** Start with `setup-guide.md`
2. **Want to understand the data?** Check out `database-overview.md`
3. **Need to run tests?** Look at `testing-guide.md`
4. **Interested in the design process?** Read `report.md`
